# ReqMan Docs

## 写作规范

+ 开新分支，MR 合入 Master
+ 新建文件用 LF，更改文件不得修改句末换行符，否则拒绝 MR
+ MR 由 项目总负责人∪前端负责人∪后端负责人 均可处理，其对应负责的部分

## 要写什么

除了这个网站作为我们最终提交的报告之外，还可以记录工作日志等等。或许是时候把飞书上的内容搬过来了。
